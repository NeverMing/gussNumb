# 🎮 猜数字游戏 - Take Number 2

> 一个简单有趣的数字推理游戏

## 🎯 游戏规则

1. **游戏目标**：猜出一个隐藏的 **4 位数字**
2. **数字规则**：
   - 可以选择是否允许数字重复
   - 不允许重复时，4 个数字各不相同
   - 允许重复时，数字可以重复出现
3. **猜测反馈**：每次猜测后，系统会告诉你**猜对了几个数字且位置正确**
   - 例如：答案是 `1234`，你猜 `1357`，反馈是 **1/4**（只有数字 1 位置和数值都对）
4. **获胜条件**：当反馈显示 **4/4** 时，表示你猜中了所有数字！

## 🎮 如何玩

1. 在输入框中输入 4 位数字（例如：`0123`）
2. 点击"猜测"按钮
3. 查看反馈结果
4. 根据反馈调整你的下一次猜测
5. 重复直到猜中答案

## 📊 游戏界面

- **左侧**：游戏区域（输入、反馈、控制按钮）
- **右侧**：历史记录（显示所有猜测和结果）
- **顶部**：计时器（记录你的游戏时间）

## 💡 游戏技巧

- 从 `0123` 开始猜测，可以测试多个数字
- 观察历史记录中的反馈，排除不可能的数字
- 开启"允许重复数字"会增加游戏难度

## 🏆 挑战

- 尝试用最少的次数猜中答案
- 尝试在最短的时间内完成游戏
- 开启重复数字模式，挑战更高难度
